# Placeholder Title
Set an alarm, now it rings \
Set for the day, ready to go \
Set the plates, breakfast for two \
Set an example, for those you care for \
Set among beautiful mountains and rivers \
Set of appointments, begrudgingly attended \
Set up a date, a much-needed escape \
Set Theory, late night homework \
Set, an opening for another band \
Set up for success
